# Simple-C-Calc Documentation

##Overview
Simple-C-Calc is a basic calculator implemented in C that can perform four arithemetic operations: addition, subtraction, multiplication and division.


## Code Structure

### 1. Including the Standard I/O Library

#include<stdio.h>
includes standard input-output library

2. main function
the entry point of the program

3. declaring variables
variables to store user input and results

4. displaying menu options
displaying available operations and takes user's choice

5. taking user input for numbers
prompts user ro enter two numbers

6. performing calculations based on user choice
performs the arithemetic operations based on user's choice and prints the result
